# Отчёт по третьей контрольной работе

## Практическое занятие №13:

Реализована регистрация Service Worker для кэширования статических ресурсов (HTML, CSS, JS) и обеспечения офлайн-доступа. Заметки сохраняются в localStorage и остаются доступными без интернета. Service Worker перехватывает сетевые запросы и возвращает ресурсы из кэша.

---

## Практическое занятие №14:

Добавлен файл manifest.json с иконками разных размеров для установки приложения на рабочий стол. Настроен favicon для отображения иконки во вкладке браузера. Добавлены мета-теги для мобильных устройств.

---

## Практическое занятие №15:

Настроен локальный HTTPS сервер через утилиту mkcert для безопасной работы Service Worker. Реализована архитектура App Shell с кэшированием каркаса приложения и динамической подгрузкой контента через innerHTML. Добавлена навигация между страницами "Главная" и "О приложении" без перезагрузки.

---

## Практическое занятие №16:

Создан Node.js сервер с Socket.IO для обмена сообщениями в реальном времени. Реализованы push-уведомления через VAPID-ключи. Добавлены кнопки включения и отключения уведомлений с управлением подписками через эндпоинты /subscribe и /unsubscribe.

---

## Практическое занятие №17:

Добавлена форма для создания заметок с напоминанием (текст + дата/время). При нажатии на кнопку "Отложить на 5 минут" в push-уведомлении время в localStorage увеличивается на 5 минут, а сервер перепланирует таймер. В списке заметок отображается иконка и время напоминания.
